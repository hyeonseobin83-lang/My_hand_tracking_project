# Technical Explanation: webcam_cv.py

이 문서는 `webcam_cv.py` 코드의 주요 기술적 특징과 구현 세부 사항을 정리합니다.

## 1. 환경 설정 및 입출력 (I/O)
*   **고해상도 설정:** `cap.set`을 통해 1280x720(HD) 해상도를 강제 지정하여 선명한 입력 영상을 확보합니다.
*   **비디오 저장 엔진:** `mp4v` 코덱을 사용하며, 실제 카메라 프레임 크기를 동적으로 읽어와(`cap.get`) `VideoWriter`에 반영합니다.
*   **실시간성 유지:** 프레임율을 20.0 FPS로 고정하여 녹화 영상의 속도를 일정하게 유지합니다.

## 2. 영상 처리 (Image Processing)
*   **좌우 반전 (Mirror Effect):** `cv2.flip(frame, 1)`을 사용하여 사용자가 거울을 보는 것과 같은 직관적인 인터페이스를 제공합니다.
*   **색상 공간 변환:** OpenCV의 BGR 형식을 MediaPipe가 인식할 수 있는 **RGB** 형식으로 변환하여 처리합니다.

## 3. AI 모델 로직 (MediaPipe)
*   **Face Detection (얼굴 감지):**
    *   상대 좌표(`0.0 ~ 1.0`)를 이미지의 실제 가로/세로 길이에 곱해 픽셀 단위 좌표로 변환하는 정규화 로직이 포함되어 있습니다.
    *   `min_detection_confidence=0.6`으로 설정하여 감지의 정확도를 조절합니다.
*   **Hand Tracking (손 추적):**
    *   `max_num_hands=2` 설정을 통해 양손을 동시에 추적할 수 있습니다.
    *   `draw_landmarks` 유틸리티를 활용해 21개의 손 마디점과 연결선을 자동으로 시각화합니다.

## 4. UI/UX 및 오버레이 (Rendering)
*   **Dynamic Overlay:** 실시간으로 변하는 `elapsed`(경과 시간)를 계산하여 `REC 00.0 / 20s` 형태의 텍스트로 화면에 렌더링합니다.
*   **가독성 확보:** 이름표 뒤에 검은색 채워진 사각형(`thickness=-1`)을 그려 텍스트가 배경에 묻히지 않도록 처리했습니다.

## 5. 흐름 제어 (Control Flow)
*   **이중 종료 조건:** 
    1. `elapsed >= DURATION`: 정해진 시간(20초)이 지나면 자동 종료.
    2. `cv2.waitKey(1) == ord('q')`: 사용자가 'q' 키를 누르면 즉시 종료.
*   **안전한 리소스 해제:** 종료 시 `release()`를 호출하여 카메라와 파일 쓰기 핸들을 정상적으로 닫아 파일 손상을 방지합니다.
